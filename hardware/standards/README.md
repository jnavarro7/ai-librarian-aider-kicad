# hardware/standards/README.md

This directory contains references to industry standards used to generate and review KiCad footprints.

Notes:
- Some standards (e.g., IPC documents) may be licensed. Do not commit copyrighted PDFs unless redistribution is explicitly permitted.
- Prefer storing links, doc identifiers, and internally-authored summaries/parameters that guide footprint generation.

Subdirectories:
- `IPC-7351/` — SMD land-pattern guidance (pad sizing, courtyard rules, density levels).
- `JEDEC/` — package outline references (mechanical dimensions and standard package families).
